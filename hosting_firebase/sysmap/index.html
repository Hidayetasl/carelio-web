📚 Protokol Açıklamaları

  • MQTT – Message Queuing Telemetry Transport: IoT için hafif publish/subscribe mesajlaşma.
  • BLE – Bluetooth Low Energy: Düşük enerji tüketimli kısa mesafe iletişim.
  • HTTP / HTTPS – Web üzerinden veri alışverişi (HTTPS güvenli sürüm).
  • TTS – Text‑to‑Speech: Yazıyı seslendirme.
  • REST – Representational State Transfer: HTTP tabanlı hafif servis mimarisi.
  • Webhook – Olay tetiklenince diğer sisteme otomatik HTTP isteği gönderme.
  • IVR – Interactive Voice Response: Çağrı merkezlerinde otomatik sesli yanıt sistemi.
